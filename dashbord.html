Résumé des instructions

  • Application enièrement en javascript. JQuery autorisé mais pas de plugin JQuery
  • Diaporama qui explique le fonctionnement
  • Google map de lyon via API, stations positionnées avec des marqueurs
  • Panneau à côté de la carte donne les détails de la station cliquée
  • API JCDecaux pour localisation et état des stations
  • Réservation de vélo, signature dans un champs libre Canvas
  • Données stockées avec Webstorage
  • Données affichées dans un pied de page en-dessous du panneau
